🚚 Supply Chain Management Analytics Dashboard | Excel, SQL, Python, Power BI

Excited to share my end-to-end Data Analyst project where I analyzed a supply chain dataset to uncover meaningful business insights and visualize key operational trends.

🧠 Project Objective
The goal of this project was to analyze supply chain data to understand:
• Product performance
• Supplier efficiency
• Regional (Location-based) sales trends
• Delivery time and operational efficiency

🛠 Tools & Technologies Used
• Microsoft Excel – Data cleaning and preparation
• SQL – Business queries and data analysis
• Python (Pandas, Matplotlib) – Exploratory Data Analysis (EDA) and visualizations
• Power BI – Interactive dashboard creation

📊 Key Work Done
• Cleaned and prepared raw dataset using Excel (removed duplicates, handled missing values, created calculated columns)
• Created Total Revenue column using Price and Number of Products Sold
• Wrote SQL queries to analyze monthly revenue, product performance, supplier contribution, and transportation modes
• Performed EDA in Python by creating multiple charts (line chart, bar chart, pie chart, histogram)
• Built an interactive Power BI dashboard with KPIs, charts, and slicers

📌 Dashboard Highlights
• Total Revenue KPI
• Average Delivery Time (derived using manufacturing + shipping time)
• Monthly Sales Trend
• Product-wise Revenue Analysis
• Supplier Performance
• Location-based Revenue Distribution
• Filters for Product, Location, and Year

💡 Key Insights
• Significant variation in revenue across products and locations
• Few products contribute to a major portion of total revenue
• Supplier performance is uneven across the dataset
• Delivery time varies depending on operational factors

🚀 What I Learned
• End-to-end data analysis workflow
• Data cleaning and transformation techniques
• Writing SQL queries for business insights
• Performing data analysis using Python
• Designing interactive dashboards in Power BI
• Converting data into actionable business insights
